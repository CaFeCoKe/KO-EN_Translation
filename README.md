# KO-EN_Translation
 Machine Translation Task를 수행하기 위해 KoBART를 한국어-영어 병렬 말뭉치로 fine-tuning하여 성능을 확인한다. 성능 검증은 테스트 데이터셋의 ROUGE score와 간단한 문장입력으로 구글 번역기와 모델의 출력을 비교해볼 것이다.
